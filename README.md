# Лабораторная работа №1 "Телефонная книга"

### Задание

Разработать приложение “Телефонная книга”. В данном приложении необходимо реализовать следующую функциональность:
1. Отображение списка контактов
2. Добавление нового контакта
3. Редактирование существующего контакта
4. Набор номера

### Описание реализации

Старт приложние начинается с загрузки контактов из файла _contacts_data.json_.

![](https://github.com/AlexandraFilyakova/swtecnnhomework/blob/develop/screenshots/Start.png)

После окончания загрузки на экране отображается список всех доступных контактов.

![](https://github.com/AlexandraFilyakova/swtecnnhomework/blob/develop/screenshots/Contacts_list.png)

На экране со списком есть возможность поиска контакта по Имени, Фамилии или Номеру телефона.

![](https://github.com/AlexandraFilyakova/swtecnnhomework/blob/develop/screenshots/Search.png)

Для старта вызова достаточно нажать на желаемый контакт.
Кнопка "Info" перенаправляет на экран делалей о контакте, где отображается информация о Имени, Фамилии, Телефоне и Электронной почте. Здесь доступна функция редактирования.

![](https://github.com/AlexandraFilyakova/swtecnnhomework/blob/develop/screenshots/Contact_details.png)

Так же в приложении реализована функция добавления в телефонную книгу нового контакта. Для этого на экране со списком надо нажать иконку создания в верхнем баре приложения.

![](https://github.com/AlexandraFilyakova/swtecnnhomework/blob/develop/screenshots/New_contact.png)


# Improvements

В данной реализации были сделаны некоторые допущения в процессе разработки. 
Ниже приведен список необходимых улучшений:

- Валидация для полей ввода Номера телефона и Электронной почты;
- Требуются существенные улучшения в архитектуре, в частности разделения на отдельные части логики загрузки данных и отображения их на экране;
- Отказаться от использования синглтона в качестве хранилища данных и инициализации его в Activity. Как вариант, использовать Dagger для инжекта инстанса объекта и Базы данных для хранения списка контактов;
- Улучшения возможностей поиска контактов;
- Написание информации в README.md на английском языке.
